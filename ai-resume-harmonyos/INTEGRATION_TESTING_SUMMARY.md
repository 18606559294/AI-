```
│  │  - TestApiServer               │   │

│  └─────────────────────────────────┘   │
└─────────────────────────────────────────┘
```

### 测试模式

**AAA 模式**
```cangjie
@Test
public func testExample(): Unit {
    // Arrange - 准备测试数据和 Mock
    let testData = TestDataFactory.createValidUser()
    this.apiServer.mockEndpoint(...)

    // Act - 执行被测试的操作
    let result = this.service.doSomething(testData)

    // Assert - 验证结果
    Assert.assertTrue(result.isSuccess())
}
```

**Mock 隔离模式**
- 使用 `TestApiServer` 模拟真实 API 响应
- 使用 `MockTokenStorage` 隔离状态存储
- 使用 `MockHttpClient` 拦截 HTTP 请求
