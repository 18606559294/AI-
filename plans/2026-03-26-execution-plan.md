## 项目概况

AI Resume 是一个完整的在线简历构建平台，技术栈：
- **后端**: FastAPI (Python 3.14) + SQLite
- **前端**: React 18 + TypeScript + Vite
- **测试**: Playwright E2E

## 关键决策

### 1. 项目位置
**决策**: 需要从 NTFS 迁移
**理由**: Unix 权限问题影响开发体验和部署
**方案**: 迁移到 ~/ai-resume 或全面 Docker 化

### 2. Git 策略
**决策**: 先清理后提交
**理由**: 大量未跟踪文件和修改需要分类处理
**方案**: 创建结构化提交，分 PR 合并

### 3. 测试策略
**决策**: E2E 测试优先
**理由**: 核心功能完整性高于单元测试覆盖率
**方案**: Playwright 全流程测试
