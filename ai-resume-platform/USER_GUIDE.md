# 用户使用指南 - 现有APK配置说明

## 📱 使用现有APK的配置步骤

由于Flutter依赖问题导致新APK编译延迟，您可以使用现有的APK（179MB）配合手动配置来实现完整功能。

---

## 🔧 配置步骤

### 第一步：安装APK

```bash
adb install -r ai_resume_app.apk
```

或者：
1. 将APK传输到手机
2. 使用文件管理器打开安装

### 第二步：配置后端服务器地址

**问题**：APK默认API地址为 `localhost:8000`，手机无法访问

**解决方法**：
1. 打开应用
2. 进入 **设置** 页面
3. 在 **服务器配置** 部分，输入后端地址：
   ```
   http://192.168.8.16:8000/api/v1
   ```
4. 点击 **保存配置**

### 第三步：选择并配置AI提供商

应用现在支持三个AI提供商，您可以根据需要选择：

#### 选项1：OpenAI（国际领先）
- **模型选择**: gpt-4, gpt-4-turbo, gpt-3.5-turbo
- **获取API密钥**: https://platform.openai.com
- **配置步骤**:
  1. 点击 **OpenAI** 按钮
  2. 输入API密钥（格式：sk-...）
  3. 选择模型（可选，默认gpt-4）
  4. 保存

#### 选项2：DeepSeek（国产高性价比）
- **模型选择**: deepseek-chat, deepseek-coder
- **获取API密钥**: https://platform.deepseek.com
- **配置步骤**:
  1. 点击 **DeepSeek** 按钮
  2. 输入API密钥
  3. 选择模型（可选，默认deepseek-chat）
  4. 保存

#### 选项3：小米MiMo（国产开源）
- **模型选择**: MiMo-V2-Flash
- **获取API密钥**: https://platform.xiaomimimo.com
- **配置步骤**:
  1. 点击 **小米AI** 按钮
  2. 输入API密钥
  3. 选择模型（可选，默认MiMo-V2-Flash）
  4. 保存

## ⚠️ 常见问题

### Q1: 配置后无法连接服务器
**A**: 检查以下几点：
1. 手机和电脑在同一局域网
2. 后端服务正在运行（`python -m app.main`）
3. 服务器地址配置正确：`http://192.168.8.16:8000/api/v1`
4. 防火墙未阻止8000端口

### Q2: AI功能显示错误
**A**:
1. 确认已配置有效的API密钥
2. 检查API密钥格式是否正确
3. 尝试切换到其他提供商测试
4. 查看后端日志：`tail -f logs/app.log`

### Q3: 生成简历时提示超时
**A**:
1. 检查网络连接
2. 尝试切换到速度更快的模型（如小米MiMo或DeepSeek）
3. 减少简历内容复杂度

### Q4: 设置页面找不到
**A**:
1. 确保使用的是最新版APK
2. 在应用首页点击左上角菜单
3. 选择"设置"选项

## 🔄 切换AI提供商

您随时可以切换AI提供商，无需重启应用：

1. 打开 **设置**
2. 点击不同的提供商按钮（OpenAI/DeepSeek/小米AI）
3. 输入对应提供商的API密钥
4. 点击 **保存配置**
5. 返回使用AI功能，自动使用新配置

**注意**：每个提供商的API密钥独立保存，切换回来时无需重新输入。

**小米MiMo**:
1. 访问 https://platform.xiaomimimo.com
2. 注册小米账号
3. 进入API-Keys页面
4. 创建API密钥

## ✅ 配置验证

配置完成后，可以通过以下方式验证：

### 方法1：查看设置页面
- 进入设置页面，确认提供商已选中
- API密钥输入框显示已保存的密钥（部分隐藏）

### 方法2：使用AI功能
1. 创建或编辑简历
2. 点击"AI生成"或"AI优化"
3. 如果正常生成，说明配置成功

### 方法3：查看后端日志
```bash
cd backend
tail -f logs/app.log | grep -i "ai\|provider"
```

应该看到类似输出：
```
INFO - Using AI provider: deepseek
INFO - Model: deepseek-chat
INFO - API call successful
```
