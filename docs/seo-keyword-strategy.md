## 五、技术实施要点

1. 每个页面的 `<title>` 包含1个核心词 + 1个长尾词
2. H1 包含核心关键词
3. Meta description 自然包含2-3个关键词
4. 图片 alt 属性包含关键词
5. 内部链接使用关键词作为锚文本
6. URL 结构保持简洁，使用拼音或英文
