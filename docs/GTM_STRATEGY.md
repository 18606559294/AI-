## 一、产品定位

### 1.1 价值主张
**"30 秒生成专业简历，AI 助你职场起飞"**

核心差异化：
- **多模型 AI 支持** - OpenAI、DeepSeek、小米 MiMo 三选一
- **国产 AI 优先** - 特别优化 DeepSeek 和小米 MiMo
- **本地数据存储** - API 密钥仅保存在本地，确保隐私安全
- **多格式导出** - PDF、Word、HTML 一键导出

### 1.2 目标用户画像

| 用户群体 | 核心痛点 | 解决方案 |
|---------|---------|---------|
| 应届毕业生 | 不知如何写简历 | AI 智能生成 + STAR 法则改写 |
| 职场跳槽者 | 需针对岗位优化 | JD 匹配分析 + 关键词优化 |
| 外企求职者 | 需要英文简历 | 多语言翻译支持 |
| 技术人员 | 项目经验难表述 | AI 技术简历专项优化 |
